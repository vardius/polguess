# 🎮 POLGUESS

Learn to read and pronounce Polish through a progressive, gamified training system.

POLGUESS teaches:

- 🟢 Polish letters & special characters (ą, ę, ć, ł, ź, ż …)
- 🟡 Digraphs & consonant clusters (cz, sz, dź, szcz …)
- 🟠 Syllable blocks
- 🔴 Real Polish words & phrases

With timer pressure, adaptive difficulty, phonetic guides in **English & Korean**, and multiple game modes.

# 🎯 Game Modes

POLGUESS offers 3 distinct ways to practice, each with its own mechanics:

## 📖 Reading Mode

- Shows progress per character.
- Adaptive difficulty — level increases only when all characters are mastered.
- Mistakes increase required repetitions.
- Type the romanization matching the Polish character or word.
- Phonetic hints available in **English** or **Korean**.

## 🎧 Listening Mode

- Infinite progressive queue (Letters → Digraphs → Syllables → Words).
- Polish character is hidden as `?`.
- Auto-plays the Polish pronunciation audio.
- One mistake = **Game Over!** Track your streak!

## 🎙️ Speaking Mode

- Validates your speech using the microphone (Web Speech API, `pl-PL`).
- Shows a Polish word on screen — say it out loud to advance.
- Must pronounce it correctly before the timer runs out.
- One mistake = **Game Over!**

# 🧩 Game Structure

### Stage 1 – Letters

Polish vowels, nasal vowels (ą, ę), and special consonants (ć, ł, ś, ź, ż, j, c, r…).

### Stage 2 – Digraphs

Consonant clusters and letter combinations (cz, sz, dz, dź, dż, szcz, rz…).

### Stage 3 – Syllables

Full Polish syllable reading (ma, pa, nie, czy, prze…).

### Stage 4 – Words

Real beginner Polish vocabulary — greetings, numbers, everyday nouns and phrases.
